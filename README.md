## 📌 Sobre o Projeto

Bem-vindo ao meu espaço na web! Este é o meu portfólio pessoal construído no formato **vCard**. Mais do que um simples currículo online, esta é uma vitrine digital interativa para apresentar meus projetos, habilidades técnicas e a minha evolução como desenvolvedor.

---

## ✨ Funcionalidades e Destaques (O que tem de legal aqui?)

* **Dragão Interativo (`interactive_dragon.js`):** Porque um portfólio tradicional é muito chato. Adicionei um *easter egg* / elemento interativo para surpreender quem navega pela página!
* **Página 404 Customizada:** Um design *monochrome* exclusivo para a página de erro, garantindo que o usuário não perca a experiência visual nem quando clica num link quebrado.
* **Loading Dinâmico:** Tela de carregamento animada configurada sob medida (`deadline-loading`).
* **Integração Externa:** Scripts para consumo de APIs de notícias (`api-news.js`), mantendo a página viva e dinâmica.
* **Galeria de Projetos:** Espaço dedicado e responsivo para exibir meus melhores trabalhos e códigos.

## 🤝 Base e Customizações
Nota: Este projeto utilizou como ponto de partida a excelente estrutura visual de um template vCard Open Source. A partir dessa base, assumi o desenvolvimento
para personalizar profundamennte a aplicação: integrei nvoas lógicas JavaScript, adaptei o design, reestruturei o CSS e adicionei a minha própria identidade.

## 🛠️ Tecnologias Utilizadas

A base do projeto utiliza um ecossistema Front-end robusto e clássico:

* **Estrutura & Estilização:** HTML5, CSS3, Bootstrap.
* **Comportamento & Interatividade:** Vanilla JavaScript, jQuery e bibliotecas de animação (Vegas.js).
* **Ícones:** Font Awesome e Ionicons.
